# PilTrace 비즈니스 플랜

**버전**: 1.0 | **작성일**: 2026-03-08 | **단계**: Phase 1 론칭 준비

---

## Executive Summary

PilTrace(필트레이스)는 AI 기반 한국어 개인 저널링 서비스입니다. 사용자가 100~2000자의 일기를 쓰면, 8개의 전문 AI 에이전트가 출판 프로세스를 거쳐 4000~5000자의 감성 에세이로 변환합니다. "숭숭이"라는 픽셀아트 원숭이 마스코트와 아날로그 스크랩북 미학이 MZ세대의 감성적 니즈를 충족합니다.

**핵심 차별점**:
- 8-에이전트 AI 파이프라인 (DeepMind "Agents' Room" 방법론, ICLR 2025 기반)
- 단순 일기 → 출판 수준 에세이 변환 (평균 3배 분량 확장)
- 아날로그 스크랩북 미학 (디지털 세대의 아날로그 향수)
- 게스트 접근 (회원가입 없이 에세이 저장/조회)

## 시장 분석

### 타겟 시장
- **1차 타겟**: MZ세대(1990-2005년생) 한국 여성, 저널링 관심층
- **시장 규모**:
  - 한국 AI 구독 앱: 2024년 기준 299% 성장
  - 감성 콘텐츠 앱 월간 지출: 1인당 평균 ₩40,000
  - 카카오 이모티콘 시장(유사 감성 상품): 연간 ₩1.2조, 결제 사용자 2.5M명

### 사용자 페르소나
1. **김지현 (26세, 직장인)**: 퇴근 후 일기 쓰고 싶지만 긴 글 어려움. AI가 에세이로 완성해주면 완성감 높음.
2. **이수진 (22세, 대학생)**: 인스타그램 "일기계" 운영. AI 에세이를 피드에 공유하고 싶음.
3. **박민영 (33세, 주부)**: 아이 육아 일기를 에세이로 보관. 추억 아카이빙 니즈.

## 제품 설명

### 핵심 기능
1. **AI 에세이 생성**: 일기 → 8-에이전트 파이프라인 → 4000~5000자 에세이
2. **4가지 필체**: 질문형 / 나열형 / 단어형 / 스토리형
3. **개인 서랍**: 에세이 아카이빙, 날짜/필체 태그
4. **게스트 접근**: 회원가입 없이 6자리+PIN으로 에세이 저장
5. **오늘의 한마디**: 완성 후 한 줄 감상 기록

### 8-에이전트 파이프라인 (기술 차별점)
```
Phase 1 (PARALLEL): 호기심 에이전트 + 깊이 에이전트 → 설문 2~6개
Phase 2 (PARALLEL): 설계사 에이전트 + 톤조율사 에이전트 → 아웃라인 + 톤가이드
Phase 3 (STREAMING): 작가 에이전트 → 4000~5000자 초고
Phase 4 (PARALLEL): 흐름편집자 + 문체감수관 → 리뷰 피드백
Phase 5 (STREAMING): 퇴고 에이전트 → 최종 에세이
```

## 수익 모델

### Phase 1 (0~1K MAU): 무료 서비스
- 전체 무료 (사용자 확보 우선)
- AI 비용: ~₩28/에세이 (GPT-5-mini)
- 월 100 DAU 기준: ~₩84,000/월 (AI 비용만)

### Phase 2 (1K~5K MAU): 프리미엄 구독 도입
| 플랜 | 월 요금 | 연 요금 | 혜택 |
|------|--------|--------|------|
| 무료 | ₩0 | ₩0 | 에세이 3편/월, 기본 필체 |
| 프리미엄 | ₩4,900/월 | ₩39,900/년 | 무제한, 전체 필체, 수정 3회 |

### Phase 3 (5K~10K MAU): 스티커 IAP 추가
- 숭숭이 스티커팩: ₩1,500~₩2,500
- 특별 에디션 배경: ₩990

### 단위 경제 (Unit Economics)
- ARPU (Average Revenue Per User): ₩4,900/월
- LTV (Lifetime Value): ₩4,900 × 20개월 = **₩98,000**
- AI 비용: ~₩28/에세이 × 15편/월 = ₩420/월/유저
- 마진율: (₩4,900 - ₩420) / ₩4,900 = **91%**
- CAC (Customer Acquisition Cost) 목표: ₩32,667 (LTV/3)
